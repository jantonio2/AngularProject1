OFERTA ACADÉMICA - TECNOLOGÍAS WEB

Introducción

En la materia de tecnologías web el estudiante aprenderá sobre las tecnologías que nos ayudan a crear sitios web, administrar estos, aprenderá sobre tecnologías de control de repositorios, patrones para tener un código limpio y sobre el uso y unión del frontend y backend.

Costo, tiempo y certificaciones por módulo

Módulo Costo Tiempo Certificación
Html y CSS 3 créditos 1 mes Desarrollador Junior en Html y estilos
Uso de Git y MVC 2 créditos 1 mes Master en control de Repositorios
Javascript y DOM 3 créditos 2 mes Desarrollador Junior en JavaScript
Angular y NodeJS 4 créditos 3 mes Desarrollador experto en Frontend y Backend

Competencias que el estudiante logrará adquirir en la materia

El estudiante al terminar el curso podrá construir cualquier tipo de web en base a Angular y NodeJS, podrá relizar el frontend de una forma esbelta y el backend en base a los modelos de control de código estará bien estructurado. Ademas de tener conocimiento sobre las tecnologías de control de repositorios.
